What is Subtitle Language Search (SLS)?

Tired of getting interested in a new show/movie only to find that it doesn't have the subtitles in the language you want? Want to see the options available before you get your hopes up? Then Subtitle Language Search is for you!

SLS is a database documenting the different languages that movies and TV shows have subtitles for. All listed media is currently available on at least 1 streaming service.

Anyone can submit new entries!

This project is part of the M&T 2022 Hackathon.
